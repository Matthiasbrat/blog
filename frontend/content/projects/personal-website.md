# Personal Website

This very website you're viewing! Built with modern web technologies for maximum performance and flexibility.

## Tech Stack

- **Framework:** Astro
- **Styling:** Vanilla CSS with custom properties
- **Content:** Markdown/MDX
- **Search:** Client-side search (Pagefind ready)
- **Comments:** Giscus (GitHub Discussions)

## Features

- Dynamic section system
- View Transitions API for SPA-like navigation
- SEO optimized with sitemap and structured data
- AI-friendly robots.txt
- Fully responsive design

## Performance

- Zero JavaScript by default
- Critical CSS inlined
- Images lazy loaded
- < 100kb total page weight
