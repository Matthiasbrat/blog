# Task Management App

A modern task management application with real-time collaboration features.

## Features

- Real-time sync across devices
- Drag-and-drop task organization
- Team workspaces
- Due date reminders
- File attachments
- Activity timeline

## Tech Stack

- **Frontend:** React, TypeScript, TailwindCSS
- **Backend:** Node.js, Express, Socket.io
- **Database:** PostgreSQL
- **Auth:** JWT with refresh tokens
- **Hosting:** Railway

## Screenshots

The dashboard provides an overview of all your tasks organized by status and priority.

## Key Learnings

- Implementing optimistic updates for better UX
- Managing WebSocket connections efficiently
- Building accessible drag-and-drop interfaces
- Database query optimization for real-time features
